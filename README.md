# Number Analyzer Pro (Java)

A menu-driven Command Line Interface (CLI) application built using Java to analyze numbers and compare two numbers using expressions, operators, and logical conditions.

---

## Features

### Menu System (do-while loop)
1. Analyze a Number
2. Compare Two Numbers
3. Exit

---

### Analyze a Number

For a given integer, the program calculates:

- Square
- Cube
- Double
- Half
- Even or Odd
- Positive or Negative
- Whether it is between 10 and 50
- Divisible by 3
- Divisible by 5
- Divisible by both 3 and 5

---

### Compare Two Numbers

- Displays the larger number (using ternary operator)
- Checks if numbers are equal
- Displays the difference between them

---

## Concepts Used

- Variables & Data Types
- Arithmetic Expressions
- Relational Operators
- Logical Operators (`&&`, `||`)
- Modulus Operator (`%`)
- Ternary Operator
- `do-while` loop
- `switch` statement
- Methods
- `Math.abs()` function
